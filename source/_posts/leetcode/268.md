### 思路
创建最大范围的vector flag。\
遍历nums中的数，标记nums中的数flag为1。\
遍历flag，为0说明没有出现。
### 代码
```
class Solution {
public:
    int missingNumber(vector<int>& nums) {
        vector<int> flag(1e4+1);
        for(int i=0;i<nums.size();i++)
            flag[nums[i]]=1;
        for(int i=0;i<=nums.size();i++)
            if(flag[i]==0)
                return i;
        return 0;
    }
};
```
